# Liveness-Detection-Using-Mediapipe
Active challenge-response liveness detection system

# How to run the repo
 - Clone the repo
 - Create a virtual env in python
   `python -m venv venv`
 - Activate the virtual environment
 - Run the command
    `pip install -r requirements.txt`
 - Run the main file
    `python face.py`
 - Make sure webcam is working
